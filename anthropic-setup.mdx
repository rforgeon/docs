# Getting an Anthropic API Key

AgentRails uses Anthropic's Claude AI model. Here's how to get your API key:

## Steps to Get Your API Key

1. **Create an Anthropic Account**
   - Visit [Anthropic's website](https://www.anthropic.com)
   - Click "Sign Up" or "Get Started"
   - Complete the registration process

2. **Access the Console**
   - Log in to your Anthropic account
   - Navigate to the API section
   - Select "Create API Key"

3. **Generate Your API Key**
   - Name your key (e.g., "AgentRails Integration")
   - Choose the appropriate access level
   - Copy your API key immediately - you won't be able to see it again!

## Configure AgentRails with Your API Key

1. In your AgentRails dashboard, go to Settings
2. Find the "Anthropic API Key" section
3. Paste your API key
4. Click Save

## Usage and Billing

- Monitor your usage in the Anthropic console
- Set up billing alerts
